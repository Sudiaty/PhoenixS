			4.main函数中只调用逻辑函数而不定义，以实现数据对象、控制函数及视图分层，降低代码耦合度，提高可读性。

			0.除用于循环体内计数变量外，名称必须具有实际意义；<br>

			2.变量名采用小驼峰命名，结构体类型数据的变量名前添加"m_&lt变量类型简写&gt"前缀，例如学生中的姓名属性定义为"char m_cName[10];"；<br>
